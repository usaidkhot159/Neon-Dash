# 🎮 Neon Dash

## 📖 About

**Neon Dash** is a browser-based arcade game where players must dodge incoming obstacles and survive for as long as possible. Featuring glowing neon visuals, smooth animations, and responsive controls, the game delivers a fun and challenging experience directly in the browser.

This project was built to improve JavaScript programming skills and understand game development concepts such as animation loops, collision detection, keyboard input, and score management.

---

## ✨ Features

- 🌟 Neon-inspired UI
- 🎮 Smooth gameplay
- ⚡ Responsive keyboard controls
- 💥 Collision detection
- 📊 Live score counter
- 🔄 Restart after Game Over
- 📱 Responsive design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/usaidkhot159/Neon-Dash.git
```

### Navigate into the project

```bash
cd Neon-Dash
```

### Run

Simply open **index.html** in your browser.

Or use **Live Server** in VS Code.
